# PRAGYA — On-Device, Offline-first AI Education Platform

PRAGYA is an on-device, privacy-first education platform that enables AI-powered tutoring and content processing without requiring constant cloud connectivity. The project includes a backend API for document ingestion, chat, analysis, speech, and other education-focused workflows.

## Key Features

- Offline-first design for on-device AI experiences
- Document ingestion and semantic search
- Interactive chat and tutoring APIs
- Speech input/output and multimedia handling
- Modular routes and services for extensibility
