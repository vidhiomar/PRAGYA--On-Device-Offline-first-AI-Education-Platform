AI-Enhanced Whiteboard (Canvas)

User visits /board page —> Canvas interface loads with tools: pen, eraser, sticky notes, text tool —> User creates content on canvas using drawing tools —> User can generate educational cards by entering prompts —> Cards contain structured educational content (title + content) —> User selects multiple cards (up to 4) —> User chooses from 4 unique card operations:
  - Summarize: Creates concise summary from selected cards
  - Action Points: Extracts actionable bullet points from selected cards
  - Flashcards: Generates Q&A flashcards for studying from selected content
  - Mind Map: Creates concept cards as visual mind map from selected content
—> AI processes selected cards using Ollama DeepSeek R1 model —> New cards are added to canvas —> User can organize, move, and manage all content on infinite canvas —> All operations run through local Ollama models without internet dependency —> Canvas supports zooming, panning, and basic drawing operations —> Content can be exported or shared —> Offline functionality ensures continued use without internet
